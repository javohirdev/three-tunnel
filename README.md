# Sozlash
Three JS uchun Node dasturini yukalang [Node.js](https://nodejs.org/en/download/).

``` bash
# Kerakli kutubxonalarni o'rnatish (faqat bir marta)
npm install

# Dasturni ishga tushurish
npm run dev

# Dasturni build qilish (dist papkasi xosil bo'ladi)
npm run build
```
